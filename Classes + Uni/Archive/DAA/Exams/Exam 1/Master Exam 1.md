## Big Theta
$\log(n^{2}) = 2\log(n)$
$\log ^{2}(n) = (\log n)^{2}$
$\log n\ge n,  \forall n \ge 1$
$\sum ^{n}_{k=1}(k) = \frac{n(n+1)}{2}$
$\sum ^{n}_{k=1}(1) = n$
$\log_{2}n\le\log_{3}n$
$k^{\log_{2}n}=n^{\log_{2}k}$
$$
\begin{gather}
For (0 < n < 1): (\log_{2} n > \log_{3} n) \\
For (n = 1): (\log_{2} n = \log_{3} n = 0) \\
For (n > 1): (\log_{2} n < \log_{3} n)
\end{gather}
$$
